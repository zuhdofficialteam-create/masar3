# مسار التحرر — تقرير المشروع الكامل

> **الدوبامين الإسلامي: من الوعي إلى التغيير**
> تقرير تصميم منتج رقمي سلوكي مبني على العلم والمنهج الإسلامي

---

## ١. فكرة المنتج (Concept)

### الاسم: **مسار التحرر**
### الشعار: *من الوعي إلى التغيير — خطوة بخطوة*

### الفكرة الأساسية

"مسار التحرر" هو دليل PDF تفاعلي يومي يُحوّل المشاهد السلبي لمحتوى التوعية بالإدمان إلى مستخدم فاعل يسير وفق منهج علمي وإسلامي مُدمَج. يعتمد على مبدأ **التدرج والاستبدال** بدلاً من الهدم المفاجئ، ويُصمَّم ليُستخدم يومياً دون الحاجة إلى أي تقنية أو برمجة.

### ما الذي يجعله مختلفاً؟

| المنتجات التقليدية | مسار التحرر |
|---|---|
| تنظير ومواعظ فقط | أدوات عملية تفاعلية |
| "توقف عن الذنب" | "استبدل بهذا البديل" |
| يُقرأ مرة واحدة | يُستخدم كل يوم |
| يُلوم الإرادة | يُعيد برمجة البيئة |
| لا يتعامل مع الانتكاسة | بروتوكول إنقاذ جاهز |

## ٢. الأساس النفسي (Psychological Model)

### ٢.١ الآليات النفسية المُستخدمة

#### Hook Model (نيل إيال)
يعمل المنتج بحلقة رباعية: **محفز → فعل → مكافأة متغيرة → استثمار**.
- **المحفز:** الشعور بالذنب أو الرغبة الملحّة
- **الفعل:** فتح صفحة محددة من الـ PDF
- **المكافأة المتغيرة:** الاكتشاف الذاتي عبر الأسئلة، وتظليل مربع نجاح جديد
- **الاستثمار:** كتابة الإجابات التي تجعل المستخدم مرتبطاً بالمنتج

#### Habit Loop — دورة العادة (Charles Duhigg)
يُفكّك المنتج الحلقة السلبية ويُعيد بناءها إيجابياً:
```
[محفز قديم] → [سلوك جديد من الأبواب الأربعة] → [مكافأة نظيفة]
```

#### Dopamine Regulation (تنظيم الدوبامين)
بدلاً من قطع مصدر الدوبامين مفاجئةً (يُسبب الانتكاسة)، يُقدّم المنتج بدائل تُفرز دوبامين نظيف: الحركة، التعلم، الصحبة، الذكر.

#### CBT — العلاج السلوكي المعرفي
أسئلة التشخيص في المرحلة الأولى تستهدف إعادة هيكلة الأنماط المعرفية المشوّهة مثل: *"أنا فاشل"* → *"هذه بيانات أتعلم منها"*.

#### Self-Determination Theory (نظرية تحديد الذات)
يُحرّك المنتج الدوافع الثلاثة الداخلية:
- **الكفاءة:** نظام النقاط اليومي يُظهر التقدم
- **الاستقلالية:** المستخدم يختار البديل المناسب له
- **الانتماء:** ذكر المحاسب الاجتماعي

### ٢.٢ كيف يكسر دائرة الإدمان؟

```
المشكلة التقليدية:
إرادة قوية → إيقاف مفاجئ → ثورة الدماغ → انتكاسة → شعور بالفشل → تراجع أشد

مسار التحرر:
تشخيص صادق → فهم الآلية → استبدال تدريجي → تحصين البيئة → استمرارية مُعززة
```

### ٢.٣ الربط بمحتوى المصدر

استُخرجت كل عناصر المنتج مباشرة من محتوى "الدوبامين الإسلامي":

| المفهوم في المحتوى | التطبيق في المنتج |
|---|---|
| "الطريق السريع في الدماغ" | خريطة الحلقة السلوكية (صفحة ٣) |
| "الأبواب الأربعة للبديل" | صفحة خطة الاستبدال (صفحة ٤) |
| "التوقف المفاجئ يزيد الرغبة" | بروتوكول التدرج + عدم قطع الصنبور |
| "التحريم بالتدرج القرآني" | منهج التغيير الأسبوعي المتراكم |
| "المشكلة في الطريق لا الذنب" | صفحة تحصين البيئة (صفحة ٦) |
| "أنت لا تسقط في الذنب بل قبله" | خريطة المناطق الخطرة |

## ٣. هيكلة الـ PDF (Architecture)

### ٩ صفحات — كل صفحة لها هدف واحد واضح

| # | عنوان الصفحة | الهدف السلوكي | النوع |
|---|---|---|---|
| ١ | الغلاف + التعريف | بناء الهوية والالتزام الأول | تعريف + كتابة |
| ٢ | التشخيص الصادق | كسر وهم "أنا أعرف مشكلتي" | استبيان + مربعات |
| ٣ | علم الدوبامين | فهم العدو = نصف المعركة | شرح + تطبيق |
| ٤ | خطة الاستبدال | من "لا" إلى "نعم لشيء أفضل" | تعهّد مكتوب |
| ٥ | المتتبّع اليومي | الاستمرارية الأسبوعية | جدول تفاعلي |
| ٦ | تحصين البيئة | سد الطريق قبل السلوك | قائمة تدقيق |
| ٧ | صفحة الإنقاذ | تحويل الانتكاسة إلى بيانات | بروتوكول طارئ |
| ٨ | متابعة ٣٠ يوماً | الرؤية البصرية للتقدم | شبكة + مراجعة |
| ٩ | الخاتمة | ترسيخ الأمل والارتباط الروحي | تحفيز + آيات |

## ٤. نظام التفاعل (Interaction System)

### ٤.١ عناصر التفاعل الرئيسية

#### Checklists يومية
قوائم بمربعات يُظلّلها المستخدم يدوياً. مصممة في ٣ مستويات:
- **مستوى ١:** عادات أساسية (الصلاة، الذكر)
- **مستوى ٢:** عادات الاستبدال (حركة، تعلم، صحبة)
- **مستوى ٣:** عادات تحصين البيئة (الهاتف، الوقت، الفضاء)

#### تمارين الوعي الذاتي (Self-Awareness)
- **خريطة المحفزات:** المستخدم يُحدد متى وأين وكيف تبدأ الرغبة
- **خريطة الفائدة الخفية:** "ما الذي يمنحك هذا الفعل فعلاً؟"
- **حلقة العادة الشخصية:** يكتبها بيده لا يقرأها جاهزة
- **خريطة المناطق الخطرة:** تحديد الزمان والمكان والشعور

#### تتبّع العادات (Habit Tracking)
- جدول أسبوعي كامل (٧ أيام × ٧ عادات = ٤٩ خانة)
- شبكة الـ ٣٠ يوماً البصرية (تُظلَّل يدوياً)
- نظام النقاط الأسبوعي مع ٤ مستويات وصف

#### أسئلة قوية مُجبِرة على التفكير
```
"ما الثمن الحقيقي الذي يكلّفك هذا الذنب روحياً، نفسياً، اجتماعياً؟"
"ماذا يمنحك هذا الفعل فعلاً؟ — لا تقل لا شيء"
"إذا كان كل شيء في حياتك كما هو، كيف تتوقع أن تتغير النتيجة؟"
"ما أول علامة تحذير تشعر بها قبل الانتكاسة؟"
```

#### المحفزات البصرية
- تصميم داكن بلوني ذهبي وفيروزي يُحفّز الهدوء والجدية
- أرقام كبيرة واضحة في جدول النقاط
- شبكة ٣٠ يوماً تُشبع حاجة "رؤية التقدم"

## ٥. رحلة المستخدم (User Journey)

### المراحل الخمس

```
┌─────────────────────────────────────────────────────────────┐
│  AWARENESS         CONFRONTATION      CONTROL               │
│  الإدراك           المواجهة           التحكم                │
│  صفحة ١-٣         صفحة ٤-٥          صفحة ٦               │
│                                                             │
│  CONTINUITY        RELAPSE HANDLING                         │
│  الاستمرارية       إدارة الانتكاسة                         │
│  صفحة ٧-٨         صفحة ٧ (طارئة)                          │
└─────────────────────────────────────────────────────────────┘
```

### التفصيل

**مرحلة الإدراك (صفحات ١-٣)**
المستخدم يُعرِّف نفسه، يُسجّل هدفه، ويواجه تشخيصاً صادقاً لمحفزاته وتكاليفه الحقيقية. ثم يفهم آلية الدوبامين ويرسم حلقته السلوكية بيده.

**مرحلة المواجهة (صفحات ٤-٥)**
المستخدم لا يقرأ حلولاً — بل يكتبها. يختار بديله الجسدي، الذهني، الاجتماعي، الروحي، ويتعهّد بكلمة واحدة في اللحظة الصعبة. ثم يبدأ تتبّعه اليومي.

**مرحلة التحكم (صفحة ٦)**
تدقيق شامل للبيئة: الهاتف، الفضاء الجسدي، جدول الوقت. ورسم خريطة المناطق الخطرة الخاصة به.

**مرحلة الاستمرارية (صفحة ٨)**
شبكة ٣٠ يوماً توفر دافعاً بصرياً مستمراً. المراجعة الأسبوعية تُحوّل التجربة إلى حكمة مُراكَمة.

**إدارة الانتكاسة (صفحة ٧)**
متاحة دائماً كصفحة طارئة. تُوقف دورة الذنب-الفشل-التخلي، وتُحوّل الانتكاسة إلى تحليل وعودة.

## ٦. آلية الاستمرارية (Retention Mechanism)

### لماذا لن يمل المستخدم؟

**١. التقدم المرئي**
الشبكة الـ ٣٠ يوماً تُشبع ما يُسمى في علم النفس "أثر إكمال الهدف" — كل مربع مُظلَّل يُطلق دوبامين صغير نظيف، مما يُعزز الاستمرار.

**٢. الاستثمار المتراكم**
كلما كتب المستخدم أكثر (إجابات، تعهدات، تحليلات)، كلما زاد ارتباطه بالمنتج. المعلومات المكتوبة بخط يده أقوى من أي محتوى مقروء.

**٣. نظام النقاط الواقعي**
٤ مستويات (بطل / متقدم / في البداية / راجع) بلا مثالية زائفة. المستوى الأدنى لا يُعاقب، بل يُوجّه فوراً.

**٤. المراجعة الأسبوعية**
سؤال واحد مختلف كل جمعة يُبقي الانخراط دون إرهاق.

**٥. الأبواب الأربعة المتنوعة**
المستخدم لا يتعامل مع أداة واحدة. له ٤ مسارات بديلة يتنقل بينها حسب ظروفه، مما يُقلل الملل ويزيد المرونة.

## ٧. سيناريو الانتكاسة (Relapse Handling)

### المشكلة التقليدية
معظم أدوات التغيير تتوقف عند "افعل العادة الجيدة". حين تحدث الانتكاسة، يجد المستخدم نفسه بلا خريطة، فتتحوّل إلى: **انتكاسة → ذنب → يأس → تخلٍّ كامل**.

### الحل في مسار التحرر: صفحة الإنقاذ الطارئة

#### بروتوكول ٥ خطوات فورية
```
١ → توقف        : لا تستمر. القطيعة الآن أهم من أي شيء.
٢ → استغفر      : "رب اغفر لي" — الآن. لا تنتظر حتى تشعر بالاستحقاق.
٣ → لا تحاكم    : لا "أنا فاشل" — بل "هذا كان اختباراً وسأعود".
٤ → حلّل        : ما الذي أوصلك هنا؟ سُدّ الثغرة.
٥ → ابدأ اليوم  : لا تنتظر يوم الاثنين. ابدأ من الصلاة القادمة.
```

#### نموذج تحليل الانتكاسة
يملأه المستخدم فور العودة:
- المحفز الذي أوصله
- البيئة التي كان فيها
- الشعور الذي سبقها
- الثغرة التي سيسدها
- التعديل الجديد في خطته

#### لماذا هذا فعّال؟
لأنه يُحوّل الانتكاسة من **"نهاية الطريق"** إلى **"بيانات للتحسين"**. وهذا بالضبط ما يفعله المعالجون السلوكيون المحترفون في CBT — يُعلّمون المريض ألا يرى الانتكاسة كفشل بل كمعلومة.

## ٩. التوصيات للتطوير المستقبلي

**المرحلة ١ — الحالية:** PDF تفاعلي يُطبع أو يُملأ رقمياً.

**المرحلة ٢ — قريبة:** نسخة Notion أو Google Docs تتيح التتبع الرقمي.

**المرحلة ٣ — متوسطة:** تطبيق موبايل بسيط يُرسل تذكيراً يومياً واحداً ويُتيح تظليل الشبكة.

**المرحلة ٤ — بعيدة:** مجتمع دعم خاص (مجموعة مغلقة) يُوفر المحاسبة الاجتماعية التي تذكر المحتوى بأهميتها.
